# Sales Data Analysis Project

## Project Overview
This project analyzes sales data using Python, Pandas, and Matplotlib.
It calculates:
- Total sales per product
- Best selling product
- Sales visualization using charts

The project is created using VS Code and uploaded to GitHub.

## Technologies Used
- Python
- Pandas
- Matplotlib
- VS Code
- Git & GitHub

## Project Structure
sales-data-analysis/
│
├── sales.csv
├── sales_analysis.py
└── README.md


## Dataset Information
The dataset file (sales.csv) contains:
- Product name
- Quantity sold
- Price per unit

Example:
Product,Quantity,Price  
Apple,10,2  
Banana,5,1  
Orange,8,3  

## How to Run This Project
### Step 1: Install Required Libraries
Open terminal and run:
pip install pandas matplotlib

### Step 2: Run the Python Program
python sales_analysis.py

## What This Program Does
1. Reads the sales CSV file
2. Calculates total sales (Quantity × Price)
3. Finds the best selling product
4. Displays a bar chart of total sales per product

## Sample Output
- Total sales per product printed in terminal
- Best selling product displayed
- Bar chart showing sales comparison
